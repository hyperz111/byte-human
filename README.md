# byte-human

Convert byte size to human-readable size

## Install

```sh
$ npm i byte-human
```

## Usage

If you use ES Module Syntax

```js
import byteHuman from 'byte-human';

byteHuman(2324606976); // => 2.32GB
```

If you use CommonJS Module Syntax

```js
const byteHuman = require('byte-human');

byteHuman(2324606976); // => 2.32GB
```
