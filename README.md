# byte-human

Convert byte size to human-readable size

## Install

```sh
$ npm install byte-human
```

## Usage

```js
import byteHuman from 'byte-human';

byteHuman(2324606976); // => 2.32 GB
```

## API

### byteHuman(bytes)

Convert byte size to human-readable size.

Param: 
- bytes: `number`

Return: `string`
